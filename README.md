# agrivision
Notre projet entreprise a pour but d’implémenter une interface web de visualisation de données. Lesdonnées sont récupérées directement sur les exploitations agricoles et seront disponible en local chez leclient. Une des valeurs du projet est que le client final (agriculteur) est maître de ses données et il endispose entièrement. Pour réaliser ce projet, nous allouons une partie conséquente du temps de projet àl’état de l’art et la recherche de l’existant. En effet, le but n’est pas nécessairement de développer unesolution de toute pièce mais plutôt, dans la mesure du possible, d’utiliser une solution déjà existante et del’adapter au projet d’AgriBIoT.
